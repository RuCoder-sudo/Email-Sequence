Автоматическая отправка последовательности писем после заполнения формы Elementor.

== Description ==

SSkin Email Sequence автоматически отправляет серию из трех email-писем пользователям, которые заполнили форму Elementor на вашем сайте.

Основные возможности:

* Отправка последовательности из 3-х писем
* Настраиваемые интервалы отправки
* Тестовый режим с быстрыми интервалами для проверки
* Обычный режим с интервалами в часах для реального использования
* Персонализация писем с переменными для имени и email
* Простая интеграция с формами Elementor без дополнительных настроек

Плагин автоматически отслеживает отправку любых форм Elementor на сайте, собирает данные пользователя и отправляет серию писем.

== Installation ==

1. Загрузите плагин в директорию `/wp-content/plugins/` или установите через панель плагинов WordPress
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Настройте содержание писем и интервалы отправки через меню 'Email Sequence' в административной панели

== Frequently Asked Questions ==

= Как плагин работает с формами Elementor? =

Плагин автоматически отслеживает отправку форм Elementor, не требуя дополнительной настройки. Он собирает email и имя пользователя из формы и добавляет их в базу подписчиков.

= Как настроить интервалы отправки писем? =

В панели администратора плагина есть два режима:
1. Тестовый режим с быстрыми интервалами (минуты) для проверки работы
2. Обычный режим с реальными интервалами (часы) для повседневного использования

= Как персонализировать содержание писем? =

В редакторе содержания писем вы можете использовать переменные {name} и {email}, которые будут заменены на реальные данные подписчика.
